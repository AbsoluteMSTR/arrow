sphinx-themes¶

These are some themes for Python Sphinx documentation projects.

Preview¶

To see how these themes look, visit http://vimalkumar.in/sphinx-themes

Download¶

Released versions are available from https://github.com/vkvn/sphinx-themes/downloads

You can also download this repository as a zip archive

Support¶

If there are problems with any of these themes, you can file a bug report at https://github.com/vkvn/sphinx-themes/issues
